# config/db

数据库常量配置

## 前端使用

```javascript
NEJ.define([
    'json!/path/to/fb-modules/config/db.json'
],function(db){
    console.log(db.USR_FRM_URS);
    // TODO something
});
```

## 后端使用

```javascript
var db = require('fb-modules').db;
console.log(db.USR_FRM_URS);
```
